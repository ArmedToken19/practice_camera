# Practice Camera

## О программе
Приложение для работы с камерой, предназначенное для тестирования функций видеозахвата, настройки кадров и практики программирования медиапотоков.

## Настройка перед запуском
Для корректной работы программы необходимо настроить конфигурационный файл.

**Шаг по переименованию:**
1. Найдите файл `settings.ini.example` в корневой папке.
2. Переименуйте его в `settings.ini`.
3. Откройте `settings.ini` и укажите ваши личные параметры (токены, порты или пути).

*Примечание: файл `settings.ini` добавлен в `.gitignore` и не попадет в репозиторий, чтобы сохранить ваши данные в секрете.*
